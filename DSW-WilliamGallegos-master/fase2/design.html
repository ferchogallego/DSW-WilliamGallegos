Propósito del sitio

Con el desarrollo de la presente actividad se pretende dar a conocer las etiquetas y su funcionabilidad dentro de HTML5. Es muy importante conocer su funcionalidad, importancia, aporte y evolución a lo largo del tiempo, teniendo en cuenta que en los últimos años han ido perdiendo fuerza, ya que algunos desarrolladores o frameworks han limitado su uso con el fin de enfocar los esfuerzos en la experiencia de usuario por medio de estilos de forma.

Definición de etiquetas HTML:





Las etiquetas HTML hacen referencia al lenguaje utilizado para el desarrollo de páginas web, este lenguaje se compone por diferentes etiquetas que el navegador interpreta y a través de las cuales se puede crear o modificar la apariencia de un documento. En términos generales las etiquetas son segmentos de código que nos permiten generar elementos HTML los cuales están conformados por atributos y contenido.
Etiquetas básicas de HTML
  • body para el contenido
  • head para información sobre el documento
  • div división dentro del contenido
  • a para enlaces
  • strong para poner el texto en negrita
  • br para saltos de línea
  • H1…H6 para títulos dentro del contenido
  • img para añadir imágenes al documento
  • ol para listas ordenadas
  • ul para listas desordenadas
  • li para elementos dentro de la lista
  • p para parágrafos
  • span para estilos de una parte del texto
